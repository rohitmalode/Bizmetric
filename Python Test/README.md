# Python Test

This folder contains Python test exercises and solutions completed as part of examinations or assessments. The questions focus on applying Python concepts to solve practical problems.  

## Contents
- Python scripts covering test questions such as:
  - **Pattern Matching** – creating patterns using loops and conditions.
  - **Dictionary Operations** – manipulating key-value data structures.
  - **Min/Max Calculations** – finding minimum and maximum values in datasets.
  - **Interest Rate Calculations** – applying Python to simple financial computations.
- Exercises are designed to evaluate understanding of core Python concepts.

## Usage
1. Open the Python `.py` files in your preferred editor or IDE.
2. Run the scripts in Python 3.x environment.
3. Review the outputs and try modifying the code to understand different scenarios.

## Notes
- These scripts are intended for practice and test preparation.
- Make sure Python 3.x is installed to run the scripts.
