
# List Comprehension & Lambda Functions

This folder contains exercises and practice questions focused on Python **list comprehensions** and **lambda functions**. It includes around 50 questions designed to strengthen problem-solving skills and understanding of these key Python features.  

## Contents
- Python scripts and solutions covering:
  - **List Comprehensions** – concise syntax for creating and manipulating lists.
  - **Lambda Functions** – anonymous functions for short, functional-style operations.
- Exercises range from beginner to intermediate level.

## Usage
1. Open the Python `.py` files in your preferred editor or IDE.
2. Run the scripts in Python 3.x environment.
3. Observe outputs and experiment with modifying the code to understand concepts better.

## Notes
- These exercises are meant for practice and skill-building in Python.
- Ensure Python 3.x is installed for running the scripts.
